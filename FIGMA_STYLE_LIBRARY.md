# Figma样式库创建指南

## 概述

本文档指导如何在 Figma 中为追漫阁项目创建完整的设计系统样式库，包括颜色、排版、组件和样式变体。

---

## Figma文件结构

```
追漫阁设计系统
├── 01 颜色系统
│   ├── 主色调
│   ├── 辅助色
│   ├── 强调色
│   ├── 中性色
│   └── 状态色
├── 02 排版系统
│   ├── 标题
│   ├── 正文
│   └── 代码
├── 03 组件库
│   ├── 按钮
│   ├── 输入框
│   ├── 卡片
│   └── 导航
└── 04 主题变体
    ├── 霓虹紫
    ├── 海洋蓝
    ├── 日落橙
    ├── 翡翠绿
    └── 樱花粉
```

---

## 01 颜色系统

### 创建颜色变量

#### 方法1：使用Figma变量面板

1. 打开 **Variables** 面板（快捷键：`Shift + I`）
2. 创建 **Collection**：`追漫阁设计系统`
3. 创建以下 **Variable Groups**：

#### 变量分组结构

```
Collection: 追漫阁设计系统
├── Group: 颜色
│   ├── Subgroup: 主色调
│   │   ├── 主色
│   │   ├── 主色亮
│   │   └── 主色暗
│   ├── Subgroup: 背景色
│   │   ├── 背景
│   │   ├── 卡片背景
│   │   ├── 输入框背景
│   │   └── 卡片悬停背景
│   ├── Subgroup: 文本色
│   │   ├── 主要文本
│   │   ├── 次要文本
│   │   ├── 禁用文本
│   │   └── 弱化文本
│   ├── Subgroup: 状态色
│   │   ├── 成功
│   │   ├── 警告
│   │   ├── 错误
│   │   └── 信息
│   └── Subgroup: 边框色
│       ├── 边框
│       ├── 边框悬停
│       └── 边框激活
```

## 02 排版系统

### 字体变量

创建以下文本样式变量：

```
Collection: 追漫阁设计系统
└── Group: 排版
    ├── 主标题 (Heading)
    │   ├── H1
    │   ├── H2
    │   ├── H3
    │   └── H4
    ├── 正文
    │   ├── Body Large
    │   ├── Body Medium
    │   ├── Body Small
    │   └── Caption
    └── 代码
        ├── Code
        └── Code Small
```

### 文本样式规格

| 样式名称 | 字体 | 大小 | 字重 | 行高 | 字间距 |
|---------|------|------|------|------|--------|
| H1 | Inter | 32px | 700 | 1.2 | -0.02em |
| H2 | Inter | 24px | 600 | 1.3 | -0.01em |
| H3 | Inter | 20px | 600 | 1.4 | 0 |
| H4 | Inter | 16px | 600 | 1.5 | 0 |
| Body Large | Inter | 16px | 400 | 1.5 | 0 |
| Body Medium | Inter | 14px | 400 | 1.5 | 0 |
| Body Small | Inter | 12px | 400 | 1.4 | 0.01em |
| Caption | Inter | 11px | 400 | 1.4 | 0.02em |
| Code | JetBrains Mono | 14px | 400 | 1.5 | 0 |
| Code Small | JetBrains Mono | 12px | 400 | 1.4 | 0 |

### Figma文本样式创建步骤

1. 选择文本对象
2. 右键 > **Create local style** (或快捷键 `Ctrl/Cmd + Alt/Opt + 5`)
3. 命名样式：`01 颜色系统/文本/H1`
4. 重复以上步骤创建所有文本样式

## 05 组件变体创建指南

### 使用Component Properties（组件属性）

#### 按钮

1. 创建按钮组件
2. 打开 **Component Properties** 面板
3. 添加以下属性：

```
按钮组件属性:
├── Instance swap (替换实例)
│   ├── Left Icon (左侧图标)
│   └── Right Icon (右侧图标)
├── Text (文本)
│   └── Button Text (按钮文本)
├── Boolean (布尔值)
│   ├── Loading (加载中)
│   └── Disabled (禁用)
└── Variant (变体)
    ├── Size (Large | Medium | Small)
    ├── Variant (Primary | Secondary | Ghost)
    └── State (Default | Hover | Active | Focus)
```

#### 输入框

1. 创建输入框组件
2. 添加以下属性：

```
输入框组件属性:
├── Instance swap (替换实例)
│   ├── Prefix Icon (前缀图标)
│   └── Suffix Icon (后缀图标)
├── Text (文本)
│   ├── Label (标签文本)
│   ├── Placeholder (占位符文本)
│   ├── Helper Text (辅助文本)
│   └── Error Text (错误文本)
├── Boolean (布尔值)
│   ├── Required (必填)
│   ├── Disabled (禁用)
│   └── Error (错误状态)
└── Variant (变体)
    ├── Size (Large | Medium | Small)
    └── State (Default | Focus | Error)
```

## 07 最佳实践

### 命名规范

- 使用结构化命名：`类别/子类别/项目`
- 使用英文命名：`Color/Primary/500`
- 保持一致性：所有组件使用相同的命名模式

### 组件组织

- 使用 **Component Sets** 管理变体
- 使用 **Component Properties** 提高复用性
- 为每个组件创建文档说明

### 版本控制

- 使用 **Figma Branches** 进行版本管理
- 创建 **Changelog** 记录变更
- 定期发布 **Release** 版本

### 协作规范

- 使用 **Comments** 进行反馈
- 使用 **Dev Mode** 导出资源
- 创建 **Prototype** 进行交互演示
