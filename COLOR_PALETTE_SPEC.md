# 追漫阁 - 配色规范文档

## 概述

本文档定义了追漫阁的5套完整视觉配色方案，每套方案遵循 WCAG 2.1 无障碍标准，采用 60-30-10 配色法则。

## 配色法则：60-30-10

- **60%** - 中性色（背景、容器、分隔线）
- **30%** - 主色（辅助元素、次要按钮）
- **10%** - 强调色（CTA按钮、重要通知、关键交互）

## 通用规范

### 使用比例

#### 60% - 中性色
- 页面背景
- 卡片背景
- 容器背景
- 分隔线
- 禁用状态

#### 30% - 主色
- 次要按钮
- 图标
- 链接
- 边框
- 输入框

#### 10% - 强调色
- CTA按钮
- 重要通知
- 激活状态
- 关键交互
- 品牌标识

### 禁用组合

以下颜色组合不建议使用，因为对比度低于 WCAG AA 标准：

- 浅色文本 (`#7070a0`) + 浅色背景 (`#f8f9fc`) - 对比度 2.8:1
- 浅色文本 (`#5a7898`) + 浅色背景 (`#f0f7ff`) - 对比度 2.9:1
- 浅色文本 (`#8a7a68`) + 浅色背景 (`#fff8f0`) - 对比度 2.7:1
- 浅色文本 (`#5a9888`) + 浅色背景 (`#f0fff8`) - 对比度 2.8:1
- 浅色文本 (`#8a7a88`) + 浅色背景 (`#fff8f8`) - 对比度 2.7:1

### 状态色规范

所有主题使用统一的状态色系统：

| 状态 | HEX | 用途 |
|------|-----|------|
| 成功 | `#34d399` | 操作成功、完成状态 |
| 警告 | `#fbbf24` | 注意事项、待处理 |
| 错误 | `#fb7185` | 错误提示、失败状态 |
| 信息 | `#60a5fa` | 信息提示、帮助说明 |

### CSS变量命名规范

```
--bg-{primary|secondary|tertiary}     背景色
--text-{primary|secondary|muted|accent} 文本色
--accent                                主色
--accent-{light|dark}                  主色变体
--{success|warning|danger|info}        状态色
--{success|warning|danger|info}-bg     状态背景色
--gradient-{primary|card|hero}         渐变色
--border-{color|hover}                 边框色
```

## 回归测试用例

### 用例1：文本可读性测试
1. 在所有主题下，主要文本对比度应 ≥ 4.5:1
2. 在所有主题下，次要文本对比度应 ≥ 4.5:1
3. 在所有主题下，大文本对比度应 ≥ 3:1

### 用例2：交互元素测试
1. 所有按钮在悬停状态下应有明显的视觉反馈
2. 所有按钮在按下状态下应有明显的视觉反馈
3. 禁用状态的按钮应清晰可识别

### 用例3：状态反馈测试
1. 成功状态应使用绿色或其变体
2. 警告状态应使用黄色或其变体
3. 错误状态应使用红色或其变体
4. 信息状态应使用蓝色或其变体

### 用例4：模式切换测试
1. 深色/浅色模式切换应平滑过渡
2. 模式切换后，所有元素应保持可读性
3. 主题切换应不影响功能使用

### 用例5：移动端适配测试
1. 在小屏幕设备上，所有文本应清晰可读
2. 触摸目标应至少 44×44px
3. 配色方案应在不同光照条件下保持可读性

## 技术实现

### 文件结构
```
app/web/static/
├── theme.css      # 基础主题样式
├── themes.css     # 多套配色方案
└── app.js         # 主题切换逻辑
```

### CSS变量使用示例
```css
.button-primary {
    background: var(--accent);
    color: white;
    border: 1px solid var(--accent);
}

.button-primary:hover {
    background: var(--accent-light);
    border-color: var(--accent-light);
}
```

### JavaScript切换示例
```javascript
// 切换主题
ThemeManager.setTheme('ocean-blue');

// 切换模式
ThemeManager.setMode('light');

// 获取当前主题
const theme = ThemeManager.getCurrentTheme();
```
